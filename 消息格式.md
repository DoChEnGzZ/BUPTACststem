## 请求发送

~~~json
{
    "//":"request是请求代码如changetemp，请各位组员把所有的request代码列在列表中"
	"request": 111
    "//":"data是数据类型，可以是数组，普通键值对"
	"data":{}
}
~~~
## 回ACK

~~~json
{
    "//":"request是请求代码如changetemp，请各位组员把所有的request代码列在列表中"
    "request":"..."
    "//":"ACK是回信息，可以将需要的信息填在后面表格中"
    "ACK":111
     "//":"data是数据类型，可以是数组，普通键值对"
	"data":{}
}
~~~
